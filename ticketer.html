⚙️ Quick Start

Install Ticketer in four simple steps.

# 1. Clone the repo
git clone https://github.com/Eltrix-Dot/Ticketer.git

# 2. Enter directory
cd Ticketer

# 3. Install requirements
pip install -r requirements.txt

# 4. Start the bot
python Main.py
