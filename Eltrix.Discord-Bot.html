🛠️ Installation

1) Requirements

  • Python 3.11+
  • Git
  • A Discord Bot Token (create one via the Discord Developer Portal)
  • Enable Privileged Gateway Intents:
    • Server Members Intent
    • Message Content Intent

2) Clone the Repository

git clone https://github.com/Eltrix-Dot/Eltrix-Discord.bot
cd Eltrix-Discord.bot

3) Install Dependencies

pip install -r requirements.txt

4) Configure the Bot

Open Eltrix-Bot_V3.1.py and edit the configuration section. 

# ==============================
# CONFIGURATION
# ==============================

TOKEN = "YOUR_BOT_TOKEN"

GUILD_ID = 123456789012345678

LOG_CHANNEL_ID = 123456789012345678

STAFF_ROLE_IDS = {
    123456789012345678,
    123456789012345679
}

OWNER_ID = 123456789012345678

5) Start the Bot

python Eltrix-Bot_V3.1.py

Tip: In Discord, enable Developer Mode to copy IDs: User Settings → Advanced → Developer Mode.
