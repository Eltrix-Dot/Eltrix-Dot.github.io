Quick Installation

Minimal steps to run v3.1 locally. (Tip: enable Developer Mode in Discord to copy IDs.)

# Clone the bot
git clone https://github.com/Eltrix-Dot/Eltrix-Discord.bot
cd Eltrix-Discord.bot
# Install dependencies
pip install -r requirements.txt
# Configure your TOKEN in Eltrix-Bot_V3.1.py, then start
python Eltrix-Bot_V3.1.py
